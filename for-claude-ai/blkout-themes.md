# BLKOUT Theme Collection

Custom themes for BLKOUT community-owned liberation platform. Each theme embodies specific aspects of our mission and values.

---

## Usage Instructions

1. **Select a theme** based on content purpose and audience
2. **Apply colors and fonts** consistently throughout artifact
3. **Reference brand guidelines** for messaging and voice
4. **Ensure accessibility** (WCAG AA minimum)

---

## Available Themes (5)

### 1. Revolutionary Red (Primary Brand Theme)
### 2. Community Gold (Warm & Welcoming)
### 3. Liberation Teal (Healing & Growth)
### 4. Trans Joy (Celebration & Visibility)

## Theme Details

### 1. Revolutionary Red

**Purpose**: Primary brand theme for bold, action-oriented content

**Color Palette**:
- Primary: `#D4261A` (BLKOUT Red) - Headers, CTAs, emphasis
- Secondary: `#264653` (Deep Forest) - Body text, grounding
- Accent: `#F4A261` (Warm Gold) - Highlights, warmth
- Background: `#FFFFFF` (White) - Clean, accessible
- Alt Background: `#FFF5F5` (Off-white with red tint)

**Typography**:
- Headers: Poppins Bold (700-800), BLKOUT Red
- Subheaders: Poppins Semibold (600), Deep Forest
- Body: Inter Regular (400), Deep Forest
- Emphasis: Inter Semibold (600), BLKOUT Red

**Best For**:
- Campaign materials
- Calls to action
- Activism content
- Urgent announcements
- Revolutionary messaging

**Emotional Tone**: Bold, urgent, powerful, unapologetic

## Theme Selection Guide

**Question**: What's the primary purpose of this material?

| Purpose | Recommended Theme |
|---------|------------------|
| Call to action / Campaign | Revolutionary Red |
| Community building / Welcome | Community Gold |
| Education / Healing | Liberation Teal |
| Trans visibility / Pride | Trans Joy |
| Policy / Vision / Serious | Abolition Dark |

## Combining Themes

For multi-section materials, you can combine themes:

**Example 1: Event Deck**
- Cover: Revolutionary Red (bold intro)
- Community sections: Community Gold
- Trans programming: Trans Joy
- Closing vision: Abolition Dark

**Example 2: Annual Report**
- Executive summary: Abolition Dark
- Community highlights: Community Gold
- Growth metrics: Liberation Teal
- Future vision: Revolutionary Red

**Rule**: Use ONE theme per section, transition intentionally

---

## Accessibility Across All Themes

**Minimum Standards**:
- Text contrast: 4.5:1 (body), 3:1 (large text/headers)
- Colorblind-safe: Test with simulators
- Alt text: Required for all images
- Readable fonts: 16px minimum for body text

**Testing Tools**:
- WebAIM Contrast Checker
- Color Oracle (colorblind simulator)
- WAVE accessibility tool

## Custom Theme Creation

Need a theme not listed? Follow this process:

1. **Define purpose**: What's the content goal?
2. **Select base colors**: Choose from BLKOUT palette
3. **Test accessibility**: Ensure contrast ratios
4. **Create mockup**: Test with sample content
5. **Get community feedback**: Share for input
6. **Document thoroughly**: Add to this collection

**Submit custom themes** to community governance for inclusion

## Platform-Specific Applications

### Presentations (PowerPoint/Keynote/Slides)
- Full theme application
- Use gradients sparingly
- Bold headers, clear body text
- BLKOUT logo on title slide

### Documents (PDFs/Reports)
- Print-safe theme variations
- High contrast for readability
- Clean layouts, generous white space
- Liberation aesthetics over corporate minimalism

### Web Artifacts
- Use CSS custom properties for theming
- Implement dark mode toggle (Abolition Dark)
- Responsive design required
- Animation: fade-in, slide-up (accessibility-safe)

### Social Media Graphics
- Simplified color palette (2-3 colors max)
- Bold typography for legibility
- Square/portrait formats
- BLKOUT branding visible
