# BLKOUT Document Template System

**Professional Communications • Strong Branding • Liberation Aesthetics**

---

## Overview

Professional document templates for BLKOUT that balance strong brand identity with the credibility needed for policy makers, researchers, and institutional audiences. These templates work primarily as digital/online publications to manage costs and environmental impact, with optional print versions for essential use.

**Core Principle**: Every document should be immediately recognizable as BLKOUT while maintaining the professionalism required for policy and research contexts.

## 1. Briefing Paper Template

**Purpose**: Policy makers, researchers, institutional audiences - concise analysis on specific topics

**Page Specs**:
- Format: A4 (210mm × 297mm) / US Letter (8.5" × 11")
- Orientation: Portrait
- Margins: 25mm (1") all sides
- Length: 2-4 pages typical
- Output: PDF optimized for screen, optional print

### Header Design

```
┌─────────────────────────────────────────────────────────┐
│                                                           │
│  [BLKOUT Logo - Red]          BRIEFING PAPER             │
│                                                           │
│  Subject: [Title of Briefing]                            │
│  Date: [Month Year]                                      │
│  Author: [Name/Department]                               │
│                                                           │
│  ═══════════════════════════════════════════════════    │
│  [Warm Gold accent line - 3pt]                           │
└─────────────────────────────────────────────────────────┘
```

**Header Elements**:
- BLKOUT logo: Top left, 40mm wide, BLKOUT Red (#D4261A)
- "BRIEFING PAPER" text: Top right, Poppins Semibold 18pt, Deep Forest (#264653)
- Subject line: Poppins Semibold 24pt, BLKOUT Red
- Metadata (date, author): Inter Regular 11pt, Deep Forest at 70% opacity
- Warm Gold separator line: 3pt solid, full width

### Content Structure

**Section 1: Executive Summary** (Grey box)
```
Background: Light grey (#F5F5F5) or very light Warm Gold tint (#FFFBF5)
Border: Left border 4pt BLKOUT Red
Padding: 15mm internal spacing
Text: Inter Regular 11pt, Deep Forest
Max length: 150-200 words
```

**Section 2: Key Points** (Numbered or bulleted)
```
Headers: Poppins Semibold 16pt, BLKOUT Red
Body text: Inter Regular 11pt, Deep Forest, 1.6 line spacing
Bullets/Numbers: BLKOUT Red or Warm Gold
Spacing: 8mm between sections
```

**Section 3: Analysis/Evidence**
```
Headers: Poppins Semibold 14pt, Deep Forest
Subheaders: Poppins Medium 12pt, BLKOUT Red
Body text: Inter Regular 11pt, Deep Forest
Pull quotes: Liberation Teal left border (4pt), italic, 12pt
Data callouts: Warm Gold background box with BLKOUT Red numbers
```

**Section 4: Recommendations**
```
Numbered list (1, 2, 3...)
Numbers: Large (24pt), BLKOUT Red, Poppins Bold
Text: Inter Semibold 12pt, Deep Forest
Each recommendation in separate visual block with subtle background
```

### Data Visualization Guidelines

**Charts and Graphs**:
- Primary color: BLKOUT Red
- Secondary colors: Warm Gold, Liberation Teal, Warm Orange
- Background: White or light grey
- Labels: Inter Semibold 10pt, Deep Forest
- Data points: BLKOUT Red with white borders
- Grid lines: Light grey (#E0E0E0), minimal

**Tables**:
- Header row: Warm Gold background, white text (Poppins Semibold 12pt)
- Alternating rows: White and very light grey (#F9F9F9)
- Borders: 1pt Deep Forest at 20% opacity
- Numbers: Inter Regular 11pt, right-aligned
- Emphasis cells: BLKOUT Red text or Liberation Teal background

**Infographics**:
- Use BLKOUT color palette exclusively
- Bold, graphic style (liberation aesthetics)
- Accessible contrast ratios (WCAG AA minimum)
- Include alt text descriptions in accessible PDF

### Report Interior Page Layout

Same header/footer system as briefing paper, but with:
- Chapter name in footer (Inter Regular 9pt, BLKOUT Red)
- More generous margins for longer reading (30mm/1.2")
- Increased line spacing (1.8) for body text
- Pull quotes styled with Liberation Teal accent

## 4. Policy Proposal Template

**Purpose**: Formal proposals to government, institutions, funding bodies

**Format**: Builds on Briefing Paper template with added formality

### Additional Sections

**Cover Letter** (if required):
- BLKOUT letterhead (described in Template 6)
- Formal address to decision maker
- Brief introduction and request
- Signature block

**Executive Summary** (Enhanced):
- Longer than briefing paper version (300-500 words)
- Clear problem statement
- Proposed solution overview
- Expected outcomes and benefits
- Cost implications if applicable

**Detailed Proposal Body**:
1. Background and Context
2. Problem Analysis (with evidence)
3. Proposed Solution (detailed)
4. Implementation Plan (timeline, resources, partnerships)
5. Expected Outcomes (measurable)
6. Budget Overview (if applicable)
7. Evaluation Framework
8. Conclusion and Call to Action

**Appendices**:
- Supporting data and research
- Letters of support from partners
- Community consultation summaries
- References and citations

### Formality Adjustments

- More conservative color use (less Warm Gold, more Deep Forest)
- Traditional page numbering (roman numerals for front matter, arabic for body)
- Formal citation style (APA, Harvard, or as required)
- Professional binding for print versions (if needed)
- Cover page includes institutional logos if co-authored

**Still Maintain**:
- BLKOUT Red for branding and emphasis
- Liberation aesthetics (don't become corporate)
- Clear, accessible language
- Strong visual hierarchy

## 5. Research Summary Template

**Purpose**: Academic audiences, researchers, evidence dissemination

**Format**: Similar to Briefing Paper but with academic conventions

### Enhanced Elements

**Abstract** (replaces Executive Summary):
- Structured: Background, Methods, Results, Conclusions
- 200-250 words
- Keywords listed at end

**Methodology Section**:
- Research design
- Data collection methods
- Sample/participants
- Analysis approach
- Ethical considerations
- Limitations

**Results Section**:
- Clear presentation of findings
- Data tables and charts (BLKOUT color palette)
- Statistical information if applicable
- Organized by research questions or themes

**Discussion Section**:
- Interpretation of findings
- Comparison with existing research
- Implications for policy and practice
- Limitations and future research directions

**References**:
- Full academic citations
- APA, Harvard, or specified style
- Hanging indent, Inter Regular 10pt
- DOIs and URLs included

### Academic Credibility + BLKOUT Branding

**Balance**:
- BLKOUT branding in header/footer
- More subdued colors in main content (Deep Forest primary)
- BLKOUT Red used sparingly for section headers only
- Data visualizations use BLKOUT palette but professionally styled
- Liberation aesthetics present but not overwhelming

**Accessibility for Academic Readers**:
- Traditional academic structure
- Clear section numbering
- Ample white space
- Footnotes or endnotes supported
- Appendices for supplementary material

## 8. Board Meeting Agenda Template

**Purpose**: Board of Directors meetings - structured agendas for governance and decision-making

**Page Specs**:
- Format: A4 / US Letter
- Orientation: Portrait
- Margins: 25mm (1") all sides
- Length: 2-3 pages typical
- Output: PDF (sent to Board 48-72 hours in advance)

### Header Design

```
┌─────────────────────────────────────────────────────────┐
│                                                           │
│  [BLKOUT Logo - Red]          BOARD MEETING              │
│                                                           │
│  Meeting Date: [Day, DD Month YYYY]                      │
│  Meeting Time: [HH:MM - HH:MM GMT]                       │
│  Location: [Physical/Virtual]                            │
│                                                           │
│  ═══════════════════════════════════════════════════    │
│  [Warm Gold accent line - 3pt]                           │
└─────────────────────────────────────────────────────────┘
```

### Standard Agenda Structure

**1. Call to Order** (Timing: 2 min)
- Chair opens meeting
- Attendance/apologies noted
- Conflicts of interest declared

**2. Approval of Previous Minutes** (5 min)
- Review minutes from [previous meeting date]
- Vote: Approve/Amend/Defer

**3. Matters Arising from Previous Minutes** (10 min)
- Action items update
- Follow-up on decisions

**4. Board Reports** (20-30 min)
- Chair's Report
- Treasurer's Report (financial overview)
- Secretary's Report (governance matters)
- Committee Reports (if applicable)

**5. Strategic Discussion Items** (30-45 min)
[List specific topics with time allocations]
- Item 5.1: [Topic] - [X min] - Discussion/Decision
- Item 5.2: [Topic] - [X min] - Discussion/Decision
- Item 5.3: [Topic] - [X min] - Discussion/Decision

**6. Operational Items** (15-20 min)
[Day-to-day matters requiring Board input]
- Item 6.1: [Topic]
- Item 6.2: [Topic]

**7. Community Governance Matters** (10 min)
- Upcoming General Member Meeting items
- Member proposals for Board consideration
- Democratic governance updates

**8. Any Other Business (AOB)** (5 min)
- Items not on agenda (urgent only)

**9. Date of Next Meeting** (2 min)
- Confirm date, time, location

**10. Adjournment**

## General Design Guidelines (All Templates)

### BLKOUT Branding Elements

**Logo Placement**:
- Top left (most templates): 40mm wide
- Centered (cover pages, slides): 60-80mm wide
- Always maintain clear space (20mm minimum)
- Use BLKOUT Red on white/light backgrounds, white on dark backgrounds

**Color Hierarchy**:
1. **BLKOUT Red (#D4261A)**: Primary brand color, headers, key emphasis
2. **Deep Forest (#264653)**: Body text, grounding elements
3. **Warm Gold (#F4A261)**: Accents, warmth, secondary emphasis
4. **Liberation Teal (#2A9D8F)**: Callouts, data highlights, hope/growth
5. **Warm Orange (#E76F51)**: Occasional accents, celebrations

**Never**:
- Use off-brand colors
- Mix too many colors on one page (maximum 3-4 from palette)
- Use low-contrast combinations (always WCAG AA minimum)

### Typography Rules

**Font Families**:
- **Poppins**: Headers, titles, emphasis (Semibold 600, Bold 700)
- **Inter**: Body text, captions, metadata (Regular 400, Medium 500, Semibold 600)

**Hierarchy**:
- Title: Poppins Bold 24-36pt
- H1: Poppins Semibold 18-24pt
- H2: Poppins Medium 14-18pt
- Body: Inter Regular 11-12pt
- Caption: Inter Regular 9-10pt

**Line Spacing**:
- Body text: 1.6-1.8 (generous for readability)
- Headers: 1.1-1.3 (tighter, more impact)
- Captions: 1.4-1.5

**Alignment**:
- Left-aligned (primary): Most readable for body text
- Centered: Headers, cover pages, quotes
- Justified: Optional for body text, but watch for rivers/gaps
- Right-aligned: Rarely (page numbers, dates)

### Page Layout Principles

**Margins**:
- Standard: 25mm (1") all sides
- Generous: 30mm (1.2") for longer documents
- Minimal: 20mm (0.8") for one-pagers with lots of content

**Grid System**:
- 12-column grid for flexible layouts
- Align elements to grid for visual order
- Use consistent spacing between elements (8mm or 16mm modules)

**White Space**:
- Not wasted space - creates breathing room
- Increases readability and professionalism
- Don't fear empty space - embrace it

**Visual Hierarchy**:
- Size: Larger = more important
- Color: BLKOUT Red = emphasis, Deep Forest = standard
- Weight: Bold = emphasis, Regular = body
- Position: Top/left = primary, bottom/right = secondary

### Accessibility Standards

**Color Contrast** (WCAG AA Minimum):
- Text on white: Deep Forest (#264653) = 10.7:1 ✅
- Headers on white: BLKOUT Red (#D4261A) = 5.9:1 ✅
- White on BLKOUT Red: 5.9:1 ✅
- Avoid Warm Gold text on white (3.2:1, too low - large text only)

**Text Size**:
- Minimum 11pt for body text (10pt acceptable for captions)
- Minimum 18pt for headers
- Presentations: Minimum 28pt for body, 48pt for headers

**Alt Text** (PDF Accessibility):
- Provide alt text for all images
- Describe charts and graphs in text
- Use semantic structure (H1, H2, lists)
- Tag PDFs properly for screen readers

**Readability**:
- Use plain language (avoid jargon where possible)
- Short sentences and paragraphs
- Active voice preferred
- Define acronyms on first use

### Print Considerations

**When Print is Necessary**:
- Essential meetings where digital won't work
- Official government submissions requiring hard copy
- Community events where digital access limited
- Archive/record copies

**Print Specifications**:
- Resolution: 300 DPI minimum
- Color mode: CMYK for printing, RGB for digital
- Bleed: 3mm if using full-bleed backgrounds
- Paper: Recycled, FSC-certified, or sustainable options
- Binding: Saddle-stitch (thin docs), perfect-bound (thick reports)

**Print Color Conversion**:
- BLKOUT Red CMYK: C:0 M:93 Y:100 K:0
- Warm Gold CMYK: C:0 M:40 Y:70 K:0
- Liberation Teal CMYK: C:70 M:0 Y:35 K:0
- Deep Forest CMYK: C:85 M:50 Y:45 K:50

Test print before large runs - colors vary by printer.

## Technical Support & Resources

### Software Recommendations

**Free/Open Source**:
- LibreOffice (Word processing, presentations)
- Google Docs/Slides (Collaborative editing)
- Scribus (Desktop publishing)
- GIMP (Image editing)
- Inkscape (Vector graphics)

**Professional (Paid)**:
- Microsoft Office (Word, PowerPoint)
- Adobe Creative Suite (InDesign, Illustrator)
- Affinity Publisher (One-time purchase alternative to InDesign)

### Font Installation

**Poppins and Inter**:
- Free from Google Fonts
- Download: fonts.google.com
- Install on your system before using templates
- Ensure SemiBold, Bold weights for Poppins
- Ensure Regular, Medium, Semibold for Inter

### Color Management

**Digital (RGB)**:
- Use hex codes provided in brand guidelines
- Test on multiple screens for consistency

**Print (CMYK)**:
- Use CMYK values provided
- Request color proofs before large print runs
- Expect slight variation from digital versions

### PDF Creation

**Recommended Settings**:
- PDF/A format for archiving
- Embed all fonts
- Compress images (balance quality and file size)
- Include metadata (title, author, keywords)
- Tag for accessibility (structure, alt text)

**Tools**:
- Adobe Acrobat Pro (professional option)
- Word/LibreOffice "Save as PDF" (basic, accessible)
- Online converters (use with caution, check privacy)

## Troubleshooting

### Common Issues

**Issue**: Fonts don't display correctly
- **Solution**: Install Poppins and Inter from Google Fonts

**Issue**: Colors look different on screen vs. print
- **Solution**: Use CMYK values for print, RGB for digital; request color proof

**Issue**: PDF file size too large
- **Solution**: Compress images, reduce resolution (72-150 DPI for screen)

**Issue**: Template structure breaks when I add content
- **Solution**: Use styles (not manual formatting), keep text within margins

**Issue**: BLKOUT logo looks pixelated
- **Solution**: Use vector SVG format, not PNG/JPG, at appropriate size

**Issue**: Document not accessible to screen readers
- **Solution**: Use heading styles (H1, H2), add alt text to images, tag PDF
